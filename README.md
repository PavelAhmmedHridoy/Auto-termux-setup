## ✨ Features

- **Dynamic Spectrum Engine** — Every word gets a unique vibrant color (live syntax highlighting)
- **JetBrains Mono Nerd Font** with ligatures
- **Beautiful Pink/Cyan/Orange** aesthetic with custom prompt
- **Modern Tools**: `eza`, `zoxide`, `zsh`, `git`
- **Smart Progress Bar** during installation
- **One-command Update** system (`auto-termux`)
- **Auto Zsh Plugins**: Autosuggestions + Syntax Highlighting
